Flask-Selenium Screenshot API

In my latest project, I developed a Flask web application, seamlessly integrating Selenium and Chrome WebDriver to create a versatile Screenshot API. The script, running Chrome in headless mode, allows users to capture website screenshots dynamically by providing parameters such as URL, width, height, and filename through a simple API endpoint. With the ability to set the viewport size and an automatic filename increment to avoid overwrites, this tool is designed for flexibility and ease of use. The global screenshot count ensures unique filenames with each capture, and the application runs as a Flask web service, making it accessible and efficient for a variety of web development and automation tasks. This Flask-Selenium Screenshot API provides a convenient solution for on-the-fly website screenshot generation.
